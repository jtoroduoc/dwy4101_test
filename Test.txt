Modifico archivos
Pasos para iniciar repositorio local
1. git init (Dentro de la carpeta del proyecto)
2. git add . (Agregar archivos base o iniciales para el proyecto)
3. git commit -m "Mensaje del cambio"
4. git push origin master (master, podría la rama de desarrollo antes de producción)
5. git branch (Muestra ramas del repositorio de versionamiento)
6. git checkout -b nombre_nueva_rama


-- Estandar Ramas Desarrollo  
- Master (Producción)
- Developed (Desarrollo)
- Feacture (Nuevos modulos / Caracteristicas)
- QA (QA para Desarrollos)
